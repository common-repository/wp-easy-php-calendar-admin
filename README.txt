Integrates the admin area for Easy PHP Calendar (http://www.easyphpcalendar.com) into the Wordpress admin area.

== Description ==
Adds the Easy PHP Calendar events administration and setup administration to the Wordpress admin area.

NOTE: This plugin does NOT add a calendar to your Wordpress installation. It is only for use with Easy PHP Calendar, a piece of software that costs $20 and is available from http://www.easyphpcalendar.com/.


== Installation ==

1. Unzip the file and upload it to your /wp-content/plugins directory
2. Activate WP Easy PHP Calendar Admin through the 'Plugins' menu in WordPress
3. Log into the setup manager of Easy PHP Calendar and create a new Event Administration user under Users
4. Log into Wordpress as an administrator and click on the new menu item called "Easy PHP Calendar"
5. Fill out the information using the Easy PHP Calendar master admin user/pass and the new user/pass you just created, and click "Update Options"
6. Click Events or Setup, and you should be automatically logged into the system.

== Frequently Asked Questions ==

= What if I don't want to save my Easy PHP Calendar usernames and passwords in the settings? =

It will still work, you will just have to log yourself into the Easy PHP Calendar admin areas manually each time you visit.
